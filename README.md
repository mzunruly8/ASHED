ASHED
Ashe: Attack • Shield • Health • Energy
Developed by @unruly357
A MzUnruly8 Production
Overview
ASHED is a minimalist grid-link combat game built with vanilla JavaScript and TailwindCSS.
Players link 3+ matching tiles to trigger combat actions:
A — Attack
S — Shield
H — Health
E — Energy
Defeat escalating enemies and survive through increasing levels of difficulty.
Victory is achieved at Level 10.
Features
7x7 drag-link grid system
Real-time path drawing
Turn-based enemy timer
Energy-gated skills
Progressive enemy scaling
Player name input
Clean responsive UI
Mobile touch support
Tech Stack
HTML
CSS (Tailwind)
Vanilla JavaScript
Canvas API for path rendering
No frameworks. No libraries beyond Tailwind.
How to Play
Enter your name.
Begin battle.
Drag across adjacent matching tiles (3+).
Survive enemy attacks.
Reach Level 10 to win.
Development Notes
ASHED was built as a focused vertical slice emphasizing:
